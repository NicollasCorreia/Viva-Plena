# Aplicativo da Clinica

Cliente desktop em Python/Tkinter para a clinica parceira.

## Como rodar

```powershell
python app.py
```

## Requisitos

- backend Django rodando
- URL da API na tela de login

## O que tem hoje

- login da clinica
- pedido de acesso a paciente
- consulta de exames liberados
- mensagens
- indicadores basicos
