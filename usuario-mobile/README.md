# App Mobile da Usuaria

Aplicativo em Expo/React Native que consome a API do projeto.

## Como rodar

```powershell
npm install
npx expo start
```

## Requisitos

- Node 20 ou superior
- backend Django rodando

## O que tem hoje

- login e cadastro
- exames
- ciclo menstrual
- historico medico
- consultas
- mensagens
- notificacoes
